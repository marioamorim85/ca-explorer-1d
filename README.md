## About

An interactive explorer for one-dimensional Cellular Automata (CA), built to visualize and study information-processing structures in CA rules.

Based on concepts from:

> Cenek, Marques-Pita & Mitchell, _"Automatic Identification of Information-Processing Structures in Cellular Automata"_ (preprint submitted to _Physica D_)

---

## Getting Started

```bash
npm install
npm run dev
```
